## Nyaralastervezo Alkalmazas
Modern, reszponziv webalkalmazas, amely idojarasi parameterek alapjan ajanl uti celokat.

## Funkciók
-  Datum es idotartam alapu keresés
-  Homerseklet szures (min/max)
-  Preferencia alapu szures (tengerpart, kultúra, természet)
-  Intelligens pontozasi rendszer
-  Reszponziv design
-  Animalt UI elemek

## Technologiak
- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
- OpenWeatherMap API (opcionális)
- Google Places API (opcionális)
